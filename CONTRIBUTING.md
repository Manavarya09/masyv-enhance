# Contributing

## Quick start
\`\`\`bash
git clone https://github.com/Manavarya09/masyv-enhance
cd masyv-enhance
cargo build
\`\`\`

## PR checklist
- [ ] Branch from \`main\` (\`feat/...\`, \`fix/...\`, \`chore/...\`)
- [ ] \`cargo test\` passes
- [ ] No leftover \`dbg!()\` / \`println!()\`
- [ ] Commit messages explain *why*

## Reporting bugs
Open an issue with steps + expected vs actual. Security: see SECURITY.md.
